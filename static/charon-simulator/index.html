Charon Rate Limiting Simulator ▼

Charon is a cooperative system used to enforce limited access to shared resources. It is designed to grant or reject access requests for a resource on behalf of a particular user. Charon uses a unique rate limiting method with configurable burst tiers. If a user hits a limit in one tier, it has the opportunity to burst up into the next tier, where it will operate for a period of time with a different limit.

This simulator allows you to configure and visualize different burst tier configurations. You can add tiers, adjust their parameters, and make requests to see how the system behaves.

Limit
The maximum number of requests that can be granted within a single window.
Window
The period of time that granted requests are recorded.
Active Time
The length of time a tier will be active.
Cooldown Time
The length of time a tier cannot be burst into after its active period ends.
